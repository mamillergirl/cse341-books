# Overview

This is a simple Rest API that allows you to GET, PUT, POST, and DELETE authors, books, and profiles. 

You can use the routes /authors, /books, and /profile
(Rest Url)[https://cse341-books.onrender.com/]

# Development

* Node V18
* JavaScript
* MongoDB
* Render
* Google APIs OAuth
